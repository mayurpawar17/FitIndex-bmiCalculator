# FitIndex - BMI Calculator

The BMI Calculator is a mobile
application designed to provide users
with a simple and intuitive way to
calculate their Body Mass Index (BMI).
The goal of this project was to create a
user-friendly interface that allows
users to quickly determine their BMI
and understand its implications on
their health.

## Features

- Calculate BMI: 
    
    The primary feature is to calculate the Body Mass Index based on the user's weight and height inputs.

- Display Result: 
    
    Display the calculated BMI value to the user.
